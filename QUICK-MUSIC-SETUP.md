3. Click "Audio Library" → "Free music"

### □ Step 3: Rename Files (2 min)

Rename downloaded files:
- First file → `focus-1.mp3`
- Second file → `focus-2.mp3`
- Third file → `break-1.mp3`
- Fourth file → `energize-1.mp3`
- Fifth file → `sleep-1.mp3`

### □ Step 4: Copy to Project (1 min)

1. Open: `d:\pomo\public\music\`
2. Copy all 5 renamed MP3 files here

### □ Step 5: Rebuild & Test (2 min)

```powershell
cd d:\pomo
npm run build
npm run dev
```

Then:
1. Open http://localhost:5173/
2. Go to Wellness Corner → Music
3. Click Play
4. 🎵 Music plays!
