## 📥 Add YouTube Audio Library (Optional)

### Quick Steps:

1. **Download Music**
   - Go to https://studio.youtube.com/ → Audio Library
   - Download free music tracks

2. **Rename Files**
   - `focus-1.mp3`, `break-1.mp3`, etc.

3. **Copy to Project**
   - Place in `d:\pomo\public\music\`

4. **Rebuild**
   ```bash
   npm run build
   npm run dev
   ```

**Full guide:** See `MUSIC-SETUP-GUIDE.md`

## 🎮 How to Use

### In Wellness Corner:
1. Click **Music** in sidebar
2. Click **Play** on any card
3. Music plays automatically!

### In Music Panel:
1. Click your profile → **Background Music**
2. Toggle music ON
3. Select playlist
4. Choose track
5. Adjust volume
6. Enjoy! 🎵

## 🔧 Troubleshooting

### No Sound?
- ✅ Check system volume
- ✅ Check browser volume
- ✅ Try different track
- ✅ Check browser console (F12)

### Track Not Loading?
- ✅ Check internet connection (for Pixabay/FMA)
- ✅ Check file exists (for YouTube files)
- ✅ Rebuild app: `npm run build`
