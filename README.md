# 🚀 POMO - Advanced Cognitive Partner

A sophisticated, AI-powered Pomodoro timer web application designed for high-performance focus and productivity. Featuring a sleek glassmorphism dark UI, integrated wellness tools, and seamless authentication.

![Pomo Banner](https://images.unsplash.com/photo-1584622650111-993a426fbf0a?auto=format&fit=crop&q=80&w=1200)

## ✨ Key Features

- **⏱️ Smart Timer Modes**: Focus, Short Break, and Long Break presets with customizable durations.
- **🔐 Multi-Method Auth**: Secure sign-in via Google OAuth, GitHub OAuth, and traditional Email.
- **🎵 Background Soundscapes**: Integrated music player with curated focus tracks and YouTube support.
- **📊 Focus Tracking**: Monitor your productivity sessions and visualize your progress.
- **🌿 Wellness Tools**: Built-in breathing exercises, eye-strain reminders, and posture checks.
- **🤖 AI Integration**: Intelligent focus optimization and cognitive partner features.
- **💬 Real-time Chat**: Secure, confidential chat functionality for collaborative focus sessions.

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3 (Custom Properties, Flex/Grid)
- **Styling**: Modern Glassmorphism, CSS Animations, Responsive Design
- **Auth**: Google Identity Services, GitHub OAuth 2.0
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pomo.git
   cd pomo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
